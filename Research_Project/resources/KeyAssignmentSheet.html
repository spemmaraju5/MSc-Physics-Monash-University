Keyboard shortcuts

Basic functions

Keys Function
Ctrl + Q close the program
Ctrl + N open a new Feynman diagram
Ctrl + L open a Feynman diagram from a ".fg" file
Ctrl + S save the Feynman diagram
Ctrl + Shift + S save the Feynman diagram as a ".fg" file
Ctrl + P print the Feynman diagram
Ctrl + E export the Feynman diagram as PS or EPS
Ctrl + I export an image of the canvas to the clipboard
Ctrl + Z undo the last edit
Ctrl + Y redo the last edit
Ctrl + C copy the selected object to the clipboard
Ctrl + V paste an object from the clipboard

View

Keys Function
S open the cheat sheet
E close/open EditFrame
G toggle grid
C toggle clipping to angles, cycling through different options: π/2, π/4, π/16
↑ + Alt increase the space between grid points
↓ + Alt decrease the space between grid points
↑ zoom in
↓ zoom out
0 + Alt reset zoom
0 + Shift reset diagram disposition
H toggle helper lines
A toggle the highlighting of the currently selected object

Other

Keys Function
Shift + click on tile remove tile from the presets
Backspace remove the selected object
Space remove all objects
M grab mode
F check for errors in Feynman diagram
I
  • a line is selected:
    → invert the line
-
  • a line is selected:
    → draw the line doubled
  • a shape is selected:
    → draw the border of the shape doubled
D
  • a line is selected:
    → toggle dashed drawing
  • a vertex is selected:
    → toggle dashed drawing of the border
  • a shape is selected:
    → toggle dashed drawing of the border
Number keys select tile with that number
→ select the tile that is right to the lastly selected tile
← select the tile that is left to the lastly selected tile
L toggle showing of label of selected object
P
  • a line is selected:
    → toggle showing of momentum arrow

Mouse scrolling

A line is selected

Keys Function
change the curvature of the line
Ctrl change the length of the wiggles as well as the amplitude and the offset (for gluons)/sharpness (for photons) of the wiggles
Alt change the thickness of the line
Caps change length and the offset (for gluons)/sharpness (for photons) of the wiggles
Caps + Ctrl change amplitude of the wiggles
Caps + Alt change the offset (for gluons)/sharpness (for photons) of the wiggles

A vertex is selected

Keys Function
change the size of the vertex
Ctrl change the thickness of the border
Alt change the rotation of the vertex

An image is selected

Keys Function
change the scale of the image
Ctrl change the rotation of the image

A shape is selected

Keys Function
change the width and height of the shape
Alt change the thickness of the line

Random shortcuts

Keys Function
E + Shift print out all the dash lengths of the lines to the console
T print out all the lengths of the lines to the console
