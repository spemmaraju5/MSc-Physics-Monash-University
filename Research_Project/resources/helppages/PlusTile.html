The "+" Tile


The left-most tile below the canvas in the main window is the so-called "+"-tile. It allows one to add all objects except for shapes to the list of tiles. They can then be accessed like any other object. They can even be included into the model file for future use using File > Save model file (as).


An object tile can be removed from the list by pressing the Shift-key and selecting the tile (clicking on it).
