EditFrame


Even though almost all of the line and vertex marker properties are accessible through keyboard shortcuts, FeynGame also provides a graphical interface to these properties, the so-called EditFrame. It opens upon pressing "e" or via the menu View > Open/Close EditFrame, and always refers to the active object.


The adjustment of geometrical parameters through EditFrame overrules the grid and all clipping settings. This means that it is always possible to change these parameters pixelwise using EditFrame.
