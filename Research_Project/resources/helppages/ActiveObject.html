Show Active Object


The selected (clicked) objects on the canvas are the active object. Object-specific editing actions (mouse and mouse-wheel movements, EditFrame) will refer to the active objects. In order to indicate which object on the canvas is the active one, FeynGame provides the Show active object feature. If it is switched on (toggle with "a", or View > Highlight active object, the active object will be marked by a moving dash pattern.
